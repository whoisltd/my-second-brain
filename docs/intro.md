# Welcome to My Second Brain

This is my second brain — a personal knowledge base where I organize, connect, and share everything I learn. From technical deep-dives in NLP and Big Data to book reviews and language notes, this is where my curiosity lives.

## Why?

As an R&D engineer, I'm constantly learning new things — reading papers, exploring frameworks, picking up concepts from books across science, economics, and fiction. But knowledge fades fast if you don't write it down.

I built this project to solve that problem:

- **Remember** — Capture ideas and concepts before they slip away.
- **Connect** — Link knowledge across different domains to see the bigger picture.
- **Share** — Make my notes useful to others who are learning the same things.

## What's inside?

This knowledge base covers a wide range of topics that I'm passionate about:

| Topic | What you'll find |
|-------|-----------------|
| **NLP** | Natural Language Processing, RAG, automated QA |
| **Big Data** | Spark, distributed systems |
| **Probability & Statistics** | Core ML concepts, statistical foundations |
| **Books** | Reviews & takeaways from science, fiction, and non-fiction |
| **Languages** | Language learning notes & lessons |
| **And more** | And more |

## How to navigate

- Use the **sidebar** on the left to browse topics.
- Use the **search bar** (Ctrl+K) to find anything quickly.
- Each page shows the **last updated date** so you know how fresh the content is.

> *"The best time to start writing things down was years ago. The second best time is now."*
